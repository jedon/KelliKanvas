# SAF Shell Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship a usable phone-first vertical slice: SAF folder pick → Home → simple Compose slideshow.

**Architecture:** Manual `KelliKanvasApp` DI owns Room, DataStore prefs, and SAF adapter reconstruction from persisted tree URIs. Compose Navigation gates Setup vs Home from selected-root presence. Slideshow walks selected SAF roots into a flat photo list and displays one bitmap at a time with a timer and prev/next/pause controls.

**Tech Stack:** Kotlin, Jetpack Compose (TV Material), Navigation Compose, Room, DataStore, existing `source/saf` adapter, JUnit/Truth/Robolectric.

**Spec:** `docs/superpowers/specs/2026-07-18-full-app-shell-design.md` (SAF shell scope).

- [ ] **Step 3: Implement**

```kotlin
enum class ShellRoute { Setup, Home }

object ShellStartup {
    fun startRoute(
        collections: List<CatalogCollection>,
        rootsByCollection: Map<String, List<SelectedRoot>>,
    ): ShellRoute {
        val hasRoot = collections.any { (rootsByCollection[it.id] ?: emptyList()).isNotEmpty() }
        return if (hasRoot) ShellRoute.Home else ShellRoute.Setup
    }

    fun activeCollectionId(
        collections: List<CatalogCollection>,
        rootsByCollection: Map<String, List<SelectedRoot>>,
    ): String? = collections.firstOrNull { (rootsByCollection[it.id] ?: emptyList()).isNotEmpty() }?.id
}
```

suspend fun build(
    adapter: SourceAdapter,
    roots: List<SelectedRoot>,
    pageSize: Int = 64,
